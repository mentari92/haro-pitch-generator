# 🔥 HARO Pitch Generator

**Generate 3 HARO pitches in 5 seconds using AI.**

Built for [OpenAI Build Week 2026](https://openai.com/build-week/) — Track: Work & Productivity.

## What is this?

HARO (Help A Reporter Out) is the best source of high-quality backlinks — Forbes, Inc, Business Insider, TechCrunch. But applying takes time.

This tool automates the hardest part: **crafting the perfect pitch.** Just paste the query and your bio, get 3 variants ready to send.

## How to Use

1. Open `index.html` in your browser
2. Enter your OpenAI API key
3. Paste HARO query from your email
4. Add your bio/expertise
5. Click generate → get 3 pitches
6. Copy & paste to HARO → Send ✅

## Quick Start

```bash
# Just open the file in a browser
open index.html
```

No installation. No build step. No server.
